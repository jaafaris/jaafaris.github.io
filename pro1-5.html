Implementation of a Matlab-based FM Receiver using RTL-SDR(Part 1)

Isfahan University of Technology

RTL-SDR serves as a versatile digital receiver in this project, offering robust insights into signal processing. The initial phase focused on establishing efficient data reception from RTL-SDR, which involved fine-tuning parameters such as center frequency, sampling rate, and buffer length to optimize performance. Once data was received, we generated the power spectral density (PSD) of the signal, providing a detailed visualization of frequency characteristics. Enhancing usability, we developed an intuitive GUI in MATLAB, enabling users to input central frequency and observation time. The GUI seamlessly outputs the PSD of the signal for the specified time and frequency. Additionally, leveraging SNR analysis, the program identifies whether the specified center frequency aligns with a radio network, offering valuable insights for signal detection and verification.

Software: Xilinx, MATLAB

Implementation of a Matlab-based FM Receiver using RTL-SDR(Part 2)

Isfahan University of Technology

To begin, we focused on ensuring accurate data reception from the RTL-SDR, which was crucial for successful demodulation. Following this, we implemented an FM demodulator using MATLAB, applying precision-focused coding techniques. For effective signal extraction, both low-pass and band-pass filters were essential components; we designed these using MATLAB's Fdatool to achieve the required accuracy. Notably, designing precise filters is pivotal in this application, as they directly influence the clarity and quality of the demodulated FM signal.

Software: MATLAB

Hardware: RTL-SDR

Course: Principles of Communication Systems
